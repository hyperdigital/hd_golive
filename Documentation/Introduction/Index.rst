Introduction
============

The GO Live checklist extension provides a backend module to track release
readiness per site. It lets editors run checklist sessions, review items,
and mark pages as pass or fail. Page tree labels show status at a glance when
an active session is set.

Key features
------------

* Checklist sessions per site, with share and close options.
* Checklist items defined in the database (global or site-specific).
* Page status tracking and page tree labels for the active session.
* Export of checklist data to PDF.
